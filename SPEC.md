# キャンプ計画アドバイザー【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Camper Plan Advisor
**バージョン**: 1.2.0（M3・M4・M6・M7・M10実装版）
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/camper-plan-advisor/

人数・予算・季節・スキルから最適なキャンプ形式と必要装備リストを提案。登録不要・完全無料でご利用いただけます。

## データ管理（M3: 履歴機能実装）

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化
- **新機能**: 診断履歴を最大5件まで自動保存、前回の結果をワンクリック復元可能
- **キー**: 
  - `camper_last_result`: 直近の診断結果
  - `camper_history`: 過去5件の診断結果履歴

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- **Chart.js（M4: グラフ可視化）** - ドーナツチャート・レーダーチャート
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 新機能（v1.1.0）

### M3: 診断履歴の自動保存・復元
- 診断完了時に結果と入力値を自動保存
- トップページに「前回の結果を復元」ボタンを表示（履歴がある場合）
- 「診断履歴を見る」で過去5件の診断日時を確認可能

### M4: Chart.js による結果の可視化
- **スコアゲージ（3つのドーナツチャート）**:
  - 予算マッチ度（紫）
  - スキルマッチ度（青）
  - 季節スコア（橙）
- **レーダーチャート**: 3項目の総合スコア比較

### M6: 目標設定・達成度トラッキング
- **🎯 目標設定ボタン**: 診断結果表示時に目標スコア設定が可能
- **達成度表示**: 予算・スキル・季節の各項目で目標に対する達成度を％表示
- **localStorage連携**: 目標値をブラウザに保存し、複数診断での進捗追跡が可能

### M7: 比較・シミュレーション機能
- **🔄 シミュレーションボタン**: 診断結果表示時に条件を変更して再計算
- **選択肢変更UI**: 5問すべての回答を変更可能なモーダルで表示
- **比較テーブル**: 現在の結果とシミュレーション結果を並べて表示、差分を色分け表示（上昇:緑、下降:赤）

### M10: 印刷・エクスポート機能
- **🖨️ 印刷ボタン**: `window.print()` で診断結果を印刷用フォーマットで出力
- **📋 コピーボタン**: 結果テキストをクリップボードにコピー
- **💾 CSVダウンロード**: 入力値・結果を CSV ファイルで保存（ファイル名: `camper-result-YYYY-MM-DD.csv`）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. **新**: スコアゲージでビジュアル確認 → 印刷・コピー・CSV保存
6. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: キャンプ計画アドバイザーは無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
